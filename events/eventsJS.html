Mouse events

Click me!
Atribut html: onclick
S'activa just cliques damunt el div.
Double click!
Atribut html: ondblclick
S'activa just cliques damunt, dues vegades, en el div
zoom (mousemove i mouseout)
Atribut html: onmouseover \ onmouseout
S'activa quan et poses damunt el div, et permeteix canviar atributs i simular, per exemple, un zoom.
Quan surts torna enrrere
Metode per canviar de color: (Sempra en tots els div) 
	
	function changeColor(x)
	{
		x.style.background = generarcolor();
	}
	
	function generarcolor()
	{
	  long=6;
	  var caracteres = "0123456789ABCDEF";
	  var color = "";
	  for (i=0; i<long; i++) color += caracteres.charAt(Math.floor(Math.random()*caracteres.length));
	  color="#"+color;
	  return color;
	}
Mouse move -
Atribut html: onmousemove
quan mous el ratoli per damunt el div Te unes coordenades que van canviant
Mouse up -
quan deixis de clicar
Atribut html: onmouseup
S'activa just amolles el boto esquerra damunt del div
Mouse down -
just cliques
Atribut html: onmousedown
S'activa just cliques amb el boto esquerra damunt del div, no fa falta amollar

Keyboard events

on Key Down

Atribut html: onkeydown
S'activa quan apretes una tecla dins el input, no fa falta amollarlo
Empra el mateix metode per canviar de color que la resta.
on Key Up

Atribut html: onkeyup
S'activa quan apretes una tecla dins el input i l'amolles, es te que amollar la tecla o no va
on Key Press

Atribut html: onkeypress
S'activa quan apretes una tecla dins el input

A tots els metodes sels hi ha passat com a parametre this, que es ell mateix per poderse modificar

El metode que s'empra en el atribut onkeypress per saber els atributs que es pot emprar d'una tecla es el seguent: 

function informacionTecla(x)
	{
		info.value = "";
		var evento = window.event || x;
 
		var mensaje = "Tipo de evento: " + evento.type + "
